Cómo usar la app

Guía rápida para conectar (Android)

Esta guía es solo para Android. Sigue estos pasos para conectar tu dispositivo:

  1. Instala la aplicación desde Google Play usando el botón de descarga.
  2. Abre la aplicación y acepta los permisos de VPN cuando se soliciten.
  3. Elige un país o servidor desde la lista.
  4. Toca el botón Conectar (uno solo toque) para establecer la conexión.
  5. Si la conexión se interrumpe, la app intentará reconectar automáticamente.
  6. Para desconectar, abre la app y toca Desconectar.

Nota: La aplicación está diseñada únicamente para Android; las instrucciones para escritorio no están disponibles.
